## 🔄 AUTOMATION WORKFLOW

### Every Day at 9:00 AM UTC:
1. **Collect Trends** - Fetches trending topics from news/social media
2. **Create Tasks** - Generates AI generation tasks for each trend
3. **Prepare Pipeline** - Queues content for AI generation

### Every Hour (24/7):
1. **Process Queue** - Takes pending generation tasks
2. **Call AI APIs** - Generates images/videos using:
   - OpenAI DALL-E (images)
   - Replicate (video generation)
3. **Store Content** - Saves to cloud storage & database
4. **Update Library** - Makes available in gallery immediately

**Result:** Your platform **grows automatically** with fresh, trending content!

## ❓ SUPPORT

### Common Issues

**Frontend won't load?**
- Make sure you're serving from a web server, not opening file directly
- Check browser console for errors
- Verify API endpoint is correct

**API returning errors?**
- Check function logs: `insforge logs function.logs`
- Verify table exists: `insforge db tables`
- Test with curl first

**Schedules not running?**
- Check active status: `insforge schedules list`
- Verify cron syntax (5-field format)
- Check logs: `insforge schedules logs [ID]`

**Upload failing?**
- Verify bucket exists: `insforge storage buckets`
- Check file size/type
- View logs: `insforge logs function.logs`
